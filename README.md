这是基于[Shiro](https://github.com/MisakaTAT/Shiro/)的QQ机器人，做得不怎么好，基本上是自娱自乐性质的产物。

想要快速在自己的服务器上运行它，可以参见[快速开始](https://github.com/catandA/catandBOT/blob/main/QuickStart_zh_cn.md)

另外需要使用任意支持反向WebSocket的OneBot协议客户端,比如[cq-http](https://github.com/Mrs4s/go-cqhttp)

另外此项目也使用了来自[spring-mirai-server](https://github.com/protobufbot/spring-mirai-server)仓库的代码

## 目前功能列表

- 文字转语音(tts)
- 无情的贴贴机器
- QQ头像戳一戳
- 美图!
- 好康的片片(噫?)
- 获取指定用户的QQ头像扔骰子
- 通过@获取指定QQ号(好没用!)
- SPDNET相关操作:
    - 用户注册
    - 更换服务器种子
    - SPDNET-Key发送邮件(尚未实装)
